# MCP Dev Summit + 97M Installs

## Key Extractions

- **Adoption milestone:** MCP (Model Context Protocol) crossed **97 million monthly SDK downloads** as of April 2026
- **First MCP Dev Summit North America:** Held in NYC, **April 2-3, 2026**
  - **95+ sessions** covering MCP implementation, tooling, and ecosystem
- **Governance:** Donated to the **Agentic AI Foundation** (under Linux Foundation) in December 2025
- **Co-founders:** Anthropic, Block, OpenAI
- **Supporting members:** Google, Microsoft, AWS
- **Protocol purpose:** Open standard for connecting AI models to external tools, data sources, and services
- **Significance:** MCP has become the de facto standard for AI agent tool use, analogous to how HTTP standardized web communication

## Relevance to Hearing + AI

- MCP enables hearing AI systems to connect to external data sources (audiograms, EHR, fitting software) via standardized protocol
- Agentic AI applications in audiology (automated fitting, remote monitoring, patient triage) would use MCP-like protocols to interact with clinical tools
- The 97M download milestone validates the AI agent ecosystem that will increasingly power hearing healthcare automation
- Linux Foundation governance ensures long-term stability — important for medical device integration where standards longevity matters
- The protocol's co-founding by Anthropic, Block, and OpenAI with Google/Microsoft/AWS support means cross-platform compatibility — hearing AI tools built on MCP would work across different AI backends
