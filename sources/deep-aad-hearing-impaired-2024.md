# Deep Learning AAD in Hearing-Impaired Listeners

## Summary

Evaluates deep convolutional neural network models for auditory attention decoding specifically in listeners with hearing impairment — a critical but understudied population for AAD research.

## Key Findings

- Deep CNN models achieve 69.8%–82.9% accuracy using 1-second classification windows
- Inter-trial strategy evaluation with EEG data from hearing-impaired listeners
- Good performance on short (1-second) EEG windows — suitable for real-world applications
- Most prior AAD studies used normal-hearing subjects; this validates with the target population

## Significance

Most AAD research uses normal-hearing young adults in labs. This paper validates that deep learning AAD works in the actual target population (hearing-impaired listeners) with clinically relevant short windows (1 second). Essential evidence for the clinical viability of neurosteered hearing aids.
