# Deep Learning Audiogram Classification

## Summary

Develops a deep learning framework for accurately classifying pure-tone audiograms across various commonly encountered audiometric patterns. Published in Clinical Otolaryngology, 2024.

## Key Contributions

- Robust DL framework for audiogram pattern classification
- Classifies audiograms into standard categories (flat, sloping, rising, cookie-bite, etc.)
- Assists clinicians without audiology expertise in audiogram interpretation
- Potential for integration into automated audiometry pipelines

## Significance

Audiogram interpretation is a specialized skill. Deep learning classification can democratize this expertise, enabling primary care physicians, community health workers, and teleaudiology platforms to accurately interpret audiometric results without specialized training.
