# Audio-Visual Speech Enhancement for Cochlear Implant Sound Coding

## Summary

Novel noise-suppressing CI system called AVSE-ECS that uses an audio-visual speech enhancement model as a pre-processing module for a deep-learning-based ElectrodeNet-CS sound coding strategy. Uses visual cues (lip reading) combined with audio for enhanced speech understanding in noise.

## Key Contributions

- AVSE-ECS: Audio-Visual Speech Enhancement + ElectrodeNet Coding Strategy
- Multimodal approach leverages lip-reading visual cues to improve speech-in-noise
- Deep learning replaces traditional CI signal processing pipeline
- End-to-end learning from audio+video to electrode stimulation patterns

## Significance

CIs users struggle significantly in noise. Adding visual information (from a camera on glasses or CI processor) could dramatically improve speech understanding by providing lip-reading cues to disambiguate noisy audio. This is one of the first complete audio-visual CI coding systems.
