# Noah ES Surpasses 5,000 Subscribers

## Key Facts
- **HIMSA's Noah ES** cloud-based hearing care practice management platform surpassed **5,000 subscribers** globally
- Updated figures (April 15, 2026): **5,400+ users** globally, **doubled year-over-year**
- Noah ES is a cloud-based evolution of the traditional Noah desktop software
- Noah is the de facto industry standard for audiometric data storage and hearing aid fitting data exchange

## Significance
- Cloud-based patient data infrastructure is a **prerequisite for AI-driven audiology at scale**
- 5,000 subscribers represents a critical mass of cloud-connected hearing care practices
- Enables centralized (or federated) analysis of audiometric data across practices
- Standardized cloud data format enables ML training on diverse patient populations
- Noah ES as cloud infrastructure could support:
  - AI-assisted fitting recommendations
  - Population-level hearing health analytics
  - Remote fitting and teleaudiology workflows
  - Federated learning across practices (without sharing raw patient data)

## Context
- HIMSA (Hearing Instrument Manufacturers' Software Association) is an industry consortium
- Noah has been the standard audiometric data format/platform for decades
- The ES (cloud) migration represents the hearing industry's data infrastructure modernization
- Complements the workforce shortage dynamic: cloud infrastructure enables one audiologist to serve patients across multiple locations
