# Adobe + Speechmatics On-Device STT

## Key Extraction

- **Partnership:** Adobe and Speechmatics deliver cloud-grade speech recognition running entirely on-device for Adobe Premiere Pro
- **Performance:** Processes 1 hour of audio in 55 seconds on-device
- **Accuracy:** Within 5% of cloud accuracy — effectively production-grade
- **vs. Whisper:** 12-16% better accuracy than OpenAI Whisper on tested benchmarks
- **Languages:** 55+ languages supported
- **Privacy:** All processing on-device — no audio leaves the machine
- **Date:** April 21, 2026

## Significance

- **On-device parity with cloud:** Demonstrates that on-device speech recognition can match cloud performance within a narrow margin — validates the on-device ML trajectory
- **Speed:** 1hr audio in 55s is ~65x real-time processing — massive throughput for on-device inference
- **Whisper benchmark:** Outperforming Whisper by 12-16% is notable since Whisper is the de facto open-source speech recognition standard
- **55+ languages:** Multilingual on-device models at this accuracy level are new
- **Model compression proof point:** Getting cloud-grade accuracy into an on-device model that runs at 65x real-time requires significant compression and optimization

## Relevance to Hearing + AI

- **Technology transfer trajectory:** If on-device speech recognition can achieve cloud parity on a laptop/desktop, the same compression techniques could push smaller speech models toward hearing aid/hearable deployment
- **Edge AI validation:** Reinforces the broader trend that cloud AI advantages are shrinking as on-device models improve — relevant for hearing aid industry's fully on-device processing approach
- **Accessibility angle:** On-device real-time captioning with 55+ languages is directly relevant to hearing accessibility (live captioning for deaf/hard-of-hearing)
- **Benchmark reference:** The 5% cloud-accuracy gap and 12-16% Whisper improvement provide reference points for evaluating hearing device speech processing models
- **Multilingual implications:** 55+ language support in a single on-device model is relevant for global hearing aid markets that need multilingual noise reduction and speech enhancement
