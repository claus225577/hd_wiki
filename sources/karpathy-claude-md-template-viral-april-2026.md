# Karpathy's CLAUDE.md Template Goes Viral

- Created by **Forrest Chang**, derived from **Andrej Karpathy's** observations about LLM coding agent pitfalls
- Encodes **4 behavioral principles** for AI coding agents to follow
- Template provides structured instructions to prevent common LLM coding failures
- **MIT license** — freely reusable and modifiable
- Addresses the gap between LLM capability and reliable coding agent behavior
- Part of a broader trend of "prompt engineering as infrastructure" — treating agent instructions as a first-class engineering artifact
