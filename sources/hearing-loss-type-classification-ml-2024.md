# Automated Hearing Loss Type Classification

## Summary

ML-based automated classification of hearing loss type (sensorineural, conductive, mixed) from pure-tone audiometry data. Published in Nature Scientific Reports, 2024.

## Key Contributions

- Automated classification: sensorineural vs. conductive vs. mixed hearing loss
- Based on standard pure-tone audiometry data (air and bone conduction thresholds)
- Validated classification accuracy against expert audiologist assessments

## Significance

Correctly classifying hearing loss type determines the treatment pathway (hearing aids for sensorineural, surgery/medical for conductive, combined for mixed). Automating this classification supports tele-audiology and screening programs where specialist audiologists aren't available.
