# ML-Based Pure-Tone Audiometry with Automated Masking

## Summary

Evaluates an improved machine learning-based pure-tone audiometry system that includes automated masking — a critical feature for clinical validity. Published in World Journal of Otorhinolaryngology, 2025.

## Key Findings

- ML-based audiometry with automated masking (previous systems lacked masking)
- Masking is essential for accurate thresholds when ears have asymmetric hearing loss
- Performance and reliability validated against conventional manual audiometry
- Addresses key limitation noted in prior ML audiometry: "unsuited for daily clinical settings, in particular for listeners with asymmetric or conductive hearing loss"

## Significance

Automated masking is the critical missing piece for ML-based audiometry to replace manual testing in clinical settings. Without proper masking, automated tests give incorrect thresholds for conductive/asymmetric losses. This paper demonstrates that ML can handle this complex clinical judgment.
