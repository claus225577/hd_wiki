## Key Facts

- ML enables personalized hearing aid gain settings beyond one-size-fits-all prescriptive formulas (e.g., NAL-NL2, DSL).
- Random forest model achieves 3.88 dB Mean Absolute Error (MAE) in predicting preferred gain — a 40% improvement over generic prescriptive fitting formulas.
- Training set of 90,000+ audiogram-to-fitting pairs drawn from real-world clinic data, enabling data-driven generalization.
- Review covers multiple ML approaches: regression trees, neural networks, collaborative filtering, and Bayesian models for preference learning.
- Key personalization dimensions: frequency gain shape, compression ratios, noise reduction strength, and directional microphone settings.
- Challenge highlighted: capturing dynamic user preferences that shift across listening environments (quiet, noisy, music).
- Outcome data (user satisfaction, word recognition scores) increasingly used as training targets rather than audiologist-set targets alone.
