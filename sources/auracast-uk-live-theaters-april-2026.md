# Auracast in UK Live Theaters

## Key Extraction

- **Venues:** Everyman Theatre (Cheltenham) and Contact (Manchester) — first UK live theater venues using Auracast for assistive listening
- **System:** Auri Bluetooth Auracast system
- **Replaces:** 20-year-old infrared assistive listening systems
- **Latency:** Sub-lip-reader-detectable latency — critical for live performance where lip-sync matters
- **Audience:** Benefits hearing loss + neurodivergent audiences
- **Accessibility improvements:**
  - No pre-booking required (unlike previous infrared systems)
  - Any seat works — no "assistive listening section" limitations
  - Handheld receivers available for non-Auracast devices (bridges the device gap)
- **Significance:**
  - First documented real-world Auracast assistive listening deployments in UK performance venues
  - Moves Auracast from "trade show demo" to "paying audience" validation
  - Replaces infrared — the dominant assistive listening technology for 20+ years — with broadcast Bluetooth
  - Sub-lip-reader latency claim is an important engineering benchmark; infrared systems had known latency issues
  - Neurodivergent benefit expands the addressable user base beyond traditional hearing loss
  - Handheld receivers solve the chicken-and-egg problem: venues can deploy Auracast before all audience members have Auracast-compatible hearing aids
  - "Any seat" removes the stigma and logistical barrier of requesting special seating
- **Date:** April 2026
