# TSF-AADNet: Short-Window Auditory Attention Decoding

## Summary

Proposes TSF-AADNet, a novel neural network framework that integrates temporal-spatial and frequency-spatial features using dual-branch architectures and advanced attention-based fusion to improve short-window auditory attention decoding performance for real-time hearing aid applications.

## Key Contributions

- Dual-branch architecture processes temporal-spatial AND frequency-spatial EEG features simultaneously
- Advanced attention-based fusion mechanism combines branch outputs
- Designed specifically for short decision windows (critical for real-time hearing aids — users cannot tolerate multi-second delays)
- Targets neuroadaptive hearing support in smart healthcare contexts

## Significance

Short-window decoding (1-2 seconds) is essential for practical neurosteered hearing aids. Most prior AAD methods required 10-30 second windows, making them impractical. TSF-AADNet pushes toward the sub-second decoding needed for seamless listening experiences.
