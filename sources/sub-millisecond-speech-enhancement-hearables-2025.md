# Sub-millisecond Latency Speech Enhancement on Hearables

## Summary

Pushes the frontier of real-time speech enhancement latency on hearable devices. Achieves mean algorithmic latency of 0.32 ms to 1.25 ms using a computationally efficient minimum-phase FIR filter for sample-by-sample processing.

## Key Contributions

- Sub-millisecond latency: 0.32–1.25 ms mean algorithmic latency
- Sample-by-sample processing using minimum-phase FIR filter
- Designed for hearable form factor (earbuds, hearing aids)
- Computationally efficient enough for edge deployment

## Significance

Sub-millisecond latency is a breakthrough for hearable speech enhancement. At these latencies, the processed audio is virtually indistinguishable in timing from direct sound, eliminating comb-filter artifacts entirely. This enables "transparent" processing where users don't perceive any delay.
